# Super Cookie Clicker

## Build Setup

``` bash
# Install dependencies
yarn

# Serve with hot reload at localhost:8080
yarn run dev

# Build for production with minification
yarn run build
```

## Test
``` bash
# Run unit test
yarn run test:unit
```

## Linter
``` bash
# Run linter
yarn run lint
```

## Instructions

Click on the big Cookie and buy building!!!
